Balanced Parenthesis  
Given a string, return true if it contains balanced parenthesis ().  

Examples  

Input  	
string:
"(x + y) - (4)"  

Output  
true  

Input  	
string:
"(((10 ) ()) ((?)(:)))"  

Output  
true  

Input  	
string:
"(50)("  

Output  
false  

Input  	
string:
""  

Output  
true
